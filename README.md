# Functional Spatial Control Charts

Este repositorio contiene el código y datos del proyecto de investigación sobre
control estadístico funcional con correlación espacial.

## Metodología
- Descomposición de Karhunen–Loève
- Distancia de Mahalanobis funcional
- Cartas de control T² y Q

## Estructura
- `R/`: funciones principales
- `scripts/`: análisis
- `data/`: datos de ejemplo
